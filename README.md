# DotFile

My dotfiles and setup scripts for macOS. Its purpose is to keep my configuration versioned.

What is included
- Homebrew: Brewfile and scripts to install and manage applications and CLI tools
- Neovim: init files, plugin configuration and helper scripts
- Obsidian: configuration snippets
- Restic: backup scripts
- skhd + yabai: window manager and hotkey configurations for a tiling workflow
- VS Code: settings and keybindings
